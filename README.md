Miracle Inc. test before interview
===

## Project initial

### Node and Npm

*  Node Version: v16.15.0
*  Npm version: 8.5.5

### Vue-Cli

    npm install -g @vue/cli

    vue create hello-world


### SCSS
    npm install -D sass-loader sass

### pug
    npm install vue-cli-plugin-pug


### tailwind

    vue add tailwind

## Usage

### Install npm package

    npm install
    
### Start project in dev

    cd miracle
    npm run serve
